Help

Landing Page

On the landing page of the website admin, you can accessthe APK download section and proceed to login with either admin or staff credentials for administrative functions.

Dashboard

In the dashboard section, upon successful login, navigate to the dashboard where you'll find a comprehensive display featuring staff/admin list, IMEI listing, user requests, requested logs, and utility or user list also have a user setting to change the password, edit information and logout.

Add Users

In the add user section, theuser possesses the capability to add users, including staff and other admin accounts, facilitating user management within the system.

List IMEI

In the list imei section including an IMEI in the list requires the user to verify its unique identifier, the primary authentication for mobile app users to access the login and signup pages, while also enabling the generation of reports for this IMEI list

User Information

In the user information section, the admin possesses the ability to edit user details and generate comprehensive reports pertaining to users information.

User Request

In the user request section, the admin has the authority approve/reject user account requests, ensuring legitimacy and authorization as staff of the PLP.

Requested Logs

In the user requested logs section, the admin can track the origin of reports generated from the mobile application, as well as generate comprehensive reports detailing the list of user-requested logs.

Sensors/Weather Values

In the sensor/weather values section, the admin can oversee both sensor and weather values, with the added functionality to generate detailed reports for analysis and tracking purposes.
